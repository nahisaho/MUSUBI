# Changelog

All notable changes to MUSUBI SDD VS Code Extension.

## [0.1.0] - 2025-12-07

### Added
- Initial release
- Steering documents tree view
- Requirements tree view with priority indicators
- Tasks tree view with status indicators
- Skills explorer (27 skills in 10 categories)
- Status bar with compliance score
- Commands: init, validate, requirements, design, tasks, sync, analyze
- Support for 7 AI platforms
- Configuration options for auto-validation and CLI path

### Features
- CLI bridge for executing MUSUBI commands
- File watchers for auto-refresh
- Workspace state management
- Output channel for logging
